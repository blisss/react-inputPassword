# react-inputPassword
基于React实现的移动端输入密码组件

# Fetures
React √
ES6 support √
webpack √

# 启动 & 开发
// clone 仓库
git clone https://github.com/jean0218/react-inputPassword.git

// 安装依赖包
npm install

// 启动
npm run start

// 跑测试用例
npm test
